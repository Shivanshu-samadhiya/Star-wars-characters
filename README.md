# Star Wars Character App

## Run locally
1. npm install
2. npm run dev
3. Open http://localhost:5173

## Features implemented
- Fetch SWAPI /people with pagination
- Character cards with Picsum images
- Modal shows details including homeworld data
- Loading & error states

## Bonus feature
- Search: Allow searching by character name (partial match).
